# Frontend

React, TypeScript, Vite, and Tailwind CSS shell for the Property Voice Agent PoC.

## Install

```bash
npm install
```

Or from the repository root:

```bash
make frontend-install
```

## Run

```bash
npm run dev
```

Or from the repository root:

```bash
make frontend-run
```

## Build

```bash
npm run build
```

Or from the repository root:

```bash
make frontend-build
```

## Configure

See [frontend/.env.example](.env.example) for the supported environment variables.

## Validation

- Production build passed successfully.
